# Gidne V2.1: Alpha Signal & EOD Wrap-up 설계 문서

> **상태:** ✅ 구현 완료 (2026-04-08)
> **구현체:** `src/components/EODBriefing.tsx`, `src/components/WatchlistNews.tsx`

## 📌 1. 문제 정의 (Problem Statement)
"터미널이 시각적으로 예뻐졌지만, 결국 하루가 끝났을 때 나에게 무슨 '행동(Actionable Insight)'을 지시하는가?"
현재 Gidne는 데이터를 실시간으로 보여주는 데는 성공했지만, 트레이더가 일일이 수치를 해석해야 합니다. 진짜 무기가 되려면 **"시장을 이긴 주도주가 무엇인지"**, **"내 관심 종목에 무슨 뉴스가 터졌는지"**를 알아서 떠먹여 주는 비서 역할이 필요합니다.

## 👥 2. 사용자 페르소나
- **파운더 본인 (Active Day/Swing Trader)**: 장 마감 직후(EOD), 내일 매매 전략을 세우기 위해 오늘 S&P 500을 가장 크게 아웃퍼폼(Outperform)한 주도 섹터와 종목을 5분 안에 파악하고 싶은 트레이더. 
- 스팸이나 불필요한 테마 잡음보다는, 본인이 추적하는 관심 종목(Watchlist)의 뉴스만 정제해서 보길 원함.

## 🚀 3. 핵심 기능 (The "Wedge" Solution)

### A. 맞춤형 뉴스 스트림 (Watchlist-linked News)
- 전체 시장 뉴스를 뿌리는 것이 아니라, 로컬 스토리지(`saveticker`)에 저장된 **워치리스트 종목들만 필터링하여 뉴스를 패칭**.
- 관심 없는 종목의 노이즈를 100% 제거한 프라이빗 뉴스 큐레이션.

### B. 어닝(Earnings) 캘린더 내장
- 거시 경제 지표(Macro Calendar) 탭 외에 '실적(Earnings)' 탭을 분리/추가.
- 이번 주 실적을 발표하는 핵심 대형주(M7 등)의 일정과 예상 가이던스를 직관적으로 표기.

### C. EOD(End of Day) 시장 주도주 리포트 (The Killer Feature)
- 장 마감 후(또는 장중 실시간), S&P 500 지수의 등락률(벤치마크)을 기준으로, 이를 압도적으로 상회(초과 수익)한 **'진짜 주도 섹터'와 'RS 리더 종목'**을 요약해 주는 자동화 위젯 추가. 
- "오늘 시장은 -1% 하락했지만, XLE(에너지) 섹터 소속의 종목들은 +2% 상승하며 강력한 자금 유입이 관찰되었습니다." 형태의 요약 리포트 제공.

## ⚙️ 4. 기술 접근법 (완전한 비전)
- **EOD 연산**: 무거운 백엔드 불필요. 프론트엔드가 이미 다운로드한 `quotes` 객체를 순회하며 `(개별 종목 상승률) - (SPY 상승률)`을 즉석에서 계산해 상위 N개를 텍스트로 요약 출력하는 '클라이언트 사이드 컴퓨팅' 사용.
- **News**: 기존에 구성된 데이터를 재활용하거나 Yahoo Finance API 플러그인을 워치리스트 심볼 배열을 인자로 던져 호출.
