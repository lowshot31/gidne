## 📌 [간단 소개] 칸

```
파편화된 시스템과 외부 API를 결합하여 기업의 생산성 병목(Toil)을 해소하는 소프트웨어 엔지니어입니다.

대기업(HD현대) 상주 환경에서 300개 이상의 엔터프라이즈 라이선스 모니터링 자동화 시스템을 개발하고, CAD/PLM 시스템의 UI/UX 유지보수를 수행하며 안정적인 엔터프라이즈급 운영 경험을 쌓았습니다. 이후 로봇 플랫폼 연동 인터랙티브 웹 앱(국립고궁박물관), LLM 오케스트레이션, 다중 외부 REST API 통합 등 현대적인 자동화 기술 스택으로 영역을 확장해왔습니다.

프론트엔드 설계(React/Astro)부터 백엔드 DB 적재, 클라우드 배포(GCP)까지 서비스의 엔드투엔드 구동 원리를 폭넓게 이해하고 있으며, 비즈니스 문제를 가장 효율적인 기술로 해결하는 것을 목표로 합니다.
```

**아래 내용을 '성과 내용' 본문란에 복사:**

```
기업용 라이선스 모니터링 및 로그 분석 파이프라인 개발 (Python, PyQt)

[상황] 300개 이상의 HD현대 그룹 내 Siemens NX 라이선스 가용 현황을 담당자 요청 시점에만 단발성으로 수동 파악하고 있어, 실시간 정보 취합 불가 및 라이선스 비용 누수가 발생하는 구조.

[성과] 직관적이고 정확한 데이터 기반의 재배치 구조를 수립하여 기존 단발성 수동 확인의 비효율성을 근본적으로 해결하고 운영 리소스 대폭 단축.

사용 기술: Python, PyQt, DB 연동, Excel 리포팅, 로그 분석
```
